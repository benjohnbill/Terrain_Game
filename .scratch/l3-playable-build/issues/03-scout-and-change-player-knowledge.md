# 03 — Scout and Change the Player’s Knowledge

**What to build:** Let the player focus an uncertain front sector, preview a
reconnaissance intent using only the viewer projection, submit it through the
Game Runtime, understand the resulting events, and see a refreshed projection
whose estimate and strategic reading have changed. This is the first complete
player command cycle through the canonical seam.

**Blocked by:** 02 — Read and Focus the Authored World.

Status: needs-info

Specification gates: Wayfinder 03, 07, 08, 10, and 12.

- [ ] Public, exact, estimated, last-seen, and hidden information follow the accepted viewer-knowledge contract.
- [ ] Command preview is a pure consumer of viewer projection and candidate intent and has no path to authoritative truth.
- [ ] A legal reconnaissance intent returns ordered presentation events and a refreshed viewer projection.
- [ ] Successful reconnaissance visibly changes the accepted estimate representation, situation judgment when warranted, and the next command preview.
- [ ] An invalid or stale intent is rejected with a reportable reason and no authoritative transition.
- [ ] Forbidden enemy truth is absent from projections, previews, events, secondary panels, and renderer inputs rather than visually concealed.
