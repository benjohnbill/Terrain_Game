# ADR 0037: War Model Build — Sealed Combat Resolution Replaces the L2 Stage-Machine

- Status: Accepted (sealed 2026-07-13)
- Relationship:
  - → Advances ADR 0031 (force-geography terrain-bound defense + reactive
    reserve: from L2-harness-only to the build's per-sector implementation).
  - → Retires the L2-harness implementation of ADR 0032 (the sector-identity
    occupation model stands as the build target; only the tournament
    scaffolding is retired).
  - Confirms ADR 0026 (atomic single-turn resolution) — records that the L2
    multi-turn siege→field→cascade→capital machine was an unlegislated
    implementation choice that *contradicted* it, not a sealed requirement.
  - Feeds ADR 0028 (L3 build-out stack direction) — the "which stack / what
    first" input for the build design sessions.
- L-trust: L1 (four-survey document synthesis) + L2 (force-geography measured
  the uniform-defense freeze, +33% decided when broken).

## Context

R14 (DESIGN-RISKS) recorded that the L2 war machine produces no decisive
climax: crisis-OFF, ZERO annihilations/match, ~77% of wars end by
stall→white-peace, and the SPEC madmovie (shield-break → decisive battle →
cascade) essentially never fires. The crisis co-analysis (2026-07-13)
established that the draw/spectacle problem is UPSTREAM in the war system, not
in the crisis ending.

A four-survey synthesis of the sealed corpus (SPEC + DESIGN + DOMAIN_MAP +
feature docs + ADRs) then asked whether the fizzle is a property of the
*sealed* war or of the L2 *implementation*. The finding: the fizzle is
generated by layers that diverge from the sealed design —

1. **Defense granularity.** The sealed model is a per-front-sector,
   four-layer defense stack (terrain × fortification × garrison ×
   commitment; ADR 0022/0031, FORMULA D6). L2 implements a per-front uniform
   aggregate (`frontG = sectors × 900`), explicitly flagged an "abstraction to
   be replaced." Force-geography measured that uniform defense is the freeze
   driver (+33% decided when broken by terrain class).
2. **Resolution shape.** ADR 0026 seals combat as an *atomic single-turn
   resolution* — no operation spans turns; campaigns are emergent chains.
   L2 runs a multi-turn `siege → field → cascade → capital` conveyor that
   accumulates `war.stalled` — an unlegislated choice that contradicts ADR
   0026 and lets a 2-turn stall exit fire before the decisive field battle.
3. **Bot war appetite.** The load-bearing motion (opportunism read — strike a
   rival whose field army is pinned elsewhere) is UNDESIGNED (SPEC_GAP ①);
   L2 uses a static ~1.7 declare gate that freezes a symmetric board.
4. **Stall exit.** The stall→white-peace auto-close (returning all occupied
   land) is explicitly BOT POLICY ONLY (CE-⑲) and never binds a human; it
   directly generates the 77% white-peace.

The sealed match-decision gate (leadership OR dominance AND unassailable, ADR
0030/0033) works exactly as designed. The sealed model is not the cause of the
fizzle. Continuing to tune L2 dials circles on top of these artifacts (the
session that reached this ADR spent days doing so). The requirement baseline
is captured in `docs/features/war-model-build/REQUIREMENTS.md`.

## Decision

1. **Retire the L2 war stage-machine.** The `siege → field → cascade →
   capital` conveyor, the per-front uniform defense aggregate, and the
   bot-policy stall→white-peace exit are measurement-era abstractions, not
   sealed design. The L2 tournament (`mockup/combat-calc/`) is no longer the
   surface on which the war model is tuned.

2. **Re-diagnose R14.** The fizzle (0 annihilation, 77% white-peace) is a
   placeholder / harness-abstraction / bot-policy artifact, not a property of
   the sealed SPEC war, and is therefore **not fixable by L2 dial tuning**.
   Whether decisive war is the *default* (vs a rarity a skilled player forces)
   is an L3 question the build must make answerable.

3. **Build the sealed war model in game code (`js/`).** The build implements,
   at minimum on the R14 fizzle path:
   - **per-front-sector four-layer defense**, terrain-distributed (A1/D7 —
     ADR 0022/0031, force-geography FG-①…⑩);
   - **atomic single-turn resolution** with the war spine (shield-break →
     decisive battle → cascade) expressed as an emergent chain, not a scripted
     multi-turn machine (A2/B2 — ADR 0026; SPEC:147);
   - **opportunism-based bot war appetite** (C1 — designing SPEC_GAP ①).

   The full requirement spine (A combat / B decision / C bots / D support,
   each with birthplace + fizzle-contamination + priority) lives in
   `war-model-build/REQUIREMENTS.md`.

4. **Reuse the sealed logic that L2 implements faithfully.** The match-decision
   gate (ADR 0030/0033), settlement acceptance arithmetic and presets,
   named-capture transfer channels (ADR 0032), rout/escape (FORMULA D10),
   uniform integration lag (ADR 0029), and contact-gating carry over as logic;
   the build ports these, not the stage-machine scaffolding around them.

5. **Crisis stays PARKED as a backstop.** The internal-uprising arc (ADR
   0035/0036) is unchanged and stays opt-in OFF; it is revisited as a lean
   turn-30 backstop (AB-③ sudden-death framing) only after decisive war is the
   build's default outcome.

## Consequences

- **L2 is retired as a tuning surface.** No further crisis/war dial sweeps on
  `tournament.js`; the deferred crisis seal-sync (SYNC-DEBT) is paid when the
  crisis un-parks in the build context, not before.
- **The build proceeds through design sessions over `REQUIREMENTS.md`**, each
  turning an undesigned/가안 P1 item into buildable spec: the opportunism read
  (C1), the atomic war spine + ❓PROPOSED vocabulary grill (A2/B2), the
  per-sector defense port (A1/D7). ADR 0028 supplies the stack / first-slice
  sequencing.
- **This ADR pays the war-model half of the "L2 fidelity-boundary" debt**
  (SYNC-DEBT, registered 2026-07-10): the "document-specified but currently
  simplified in L2" list for the war model is `REQUIREMENTS.md`.
- **The R14 fizzle is addressed at its source** (the artifacts) rather than
  masked by bot policy or crisis dials; whether decisive war becomes the
  default is then an honest L3/playtest reading, no longer bot-hostage.
- **Risk:** the build is a larger unit of work than an L2 dial sweep, and its
  own undesigned pieces (opportunism read, roshambo layer, capital wiring)
  each need a design pass. This ADR sets the direction; it does not pre-decide
  those designs.
