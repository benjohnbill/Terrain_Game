# ADR 0005: Hybrid Historical-Fictional Place Names

Date: 2026-06-29

Status: Accepted

## Context

The game should evoke the immersion of East Asian historical strategy settings
without becoming a strict historical simulation. Realistic place naming helps
players infer terrain, culture, and strategic meaning, but literal historical
names can create unnecessary accuracy constraints and reduce future expansion
freedom.

The user may add detailed place names later, so the naming system should leave
room for extension.

## Decision

Use a hybrid naming policy.

Large geography and regional archetypes may use historically legible names or
near-historical terms, such as central plains, Guanzhong-like passes,
Jiangnan-like grain regions, Liaodong-like frontiers, or Shu-like basins.

Specific named provinces should generally use fictional East Asian-style names
that imply geography, economy, or strategic role without requiring exact
historical correspondence.

Player-facing names should read like natural place names or geographic
descriptions, not design labels. For example, use names in the spirit of
"Guanzhong Gate" or "Jiangnan Grain Belt" rather than meta phrases equivalent
to "Guanzhong-style gate" or "Jiangnan-like grain region." Words such as
"-inspired" and "-like" belong in design documentation, not in the displayed
province names.

Province descriptions should pair names with terrain and identity:

- name,
- terrain or geographic feature,
- economic/political specialty,
- strategic role.

## Consequences

The map can feel familiar and historically grounded while remaining flexible.

The project can add or rename detailed provinces later without breaking the
world model.

Documentation and UI should avoid overclaiming historical accuracy. The world
is inspired by history, not a direct recreation.
