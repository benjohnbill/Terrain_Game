# Terrain-Cradle Glossary — Tier 1

Birthplace and single definition point of terrain-cradle vocabulary
(Vocabulary Law, `.claude/rules/documentation-law.md`). Header format:
English canonical (한국어 표시어). Status ≡ DOMAIN_MAP dictionary
(AGREED/PROPOSED). Values cited from rulings live in `RULINGS.md`;
door widths' sealed home is combat-formula `MAGNITUDE.md` M11 (this
file never restates those numbers normatively).

| Term | Definition | Status |
|---|---|---|
| Void terrain (공백 지형) | Sea expressed as land: non-sector hexes no one owns; movement, ownership and vision identical to sea. Exactly one kind of "cannot cross" exists in the world. | AGREED 2026-07-07 · TC-⑧ |
| Great range (대산맥) | The void mountain mass filling the 서역–초원 gap, sea-anchored north, dying south into the Hexi corridor. | AGREED 2026-07-07 · TC-⑨ |
| Hexi corridor (하서회랑) | The hills-class door between 서역 and 하북 — the saddle between the great range and the 관중 rim; everything entering or leaving 서역 by the north passes here. | AGREED 2026-07-07 · TC-⑨ |
| Western Ring (대환) | The continuous sea-side void wall around the left hemisphere's outer rim (range→서역→관중→촉); doors read as notches. World-cause of the desert, the basins, and the range. | AGREED 2026-07-07 · TC-⑨ |
| Taishan (태산) | The 4-hex sacred massif at the 하북/한경/중원/초원 quad junction — the only mountain outside the wall system; structurally holds both diagonal non-adjacencies. | AGREED 2026-07-07 · TC-⑩ |
| Parity start (동일 시작 인구) | Every region starts at Σ population 6.0 / cap 3,600; divergence only via play. Generalized beyond population → Derived asymmetry (TC-⑭). | AGREED 2026-07-07 · TC-① |
| Derived asymmetry (파생 비대칭) | All playable state (population, fortification, garrison, investment) starts uniform across realms; the only permitted starting inequality is one derived from the authored map (geometry + fog) — e.g. 동남해 projectable-mass ceiling, treasury from economy — never baked per-realm. Declares SPEC Core Principle #8; seals fort/garrison/investment uniformity beyond TC-①'s population. | AGREED 2026-07-08 · TC-⑭ |
| Economy ladder (경제 사다리) | Regional economy totals derived as the residual that closes each geography package into a fair bet: 0.55 + 0.45 × (inbound flow ÷ avg) + projection credit. Structure agreed; constants and held terms are open grill material. | AGREED-structure 2026-07-07 · TC-③ (values 가안) |
| Inbound flow (침입 유량) | The safety metric: Σ over borders of what a neighbor can push per campaign (open = full parity field; choked = door × chokeFlow). | AGREED 2026-07-07 · TC-③ |
| Border class (국경 등급) | A border's terrain kind sets its door (open / forest / hills(new, 가안 1,300) / pass / river / strait; door widths' sealed home: MAGNITUDE M11) AND (TC-⑬) its L2 combat terrain/water/choke — the crossing keys defense to geography (binding home: RULINGS TC-⑬; multiplier homes: M5 / ADR 0015). | AGREED 2026-07-07 · TC-④; combat binding TC-⑬ 2026-07-08 |
| Corridor authoring (회랑 저작) | A partial natural barrier on an open border (부분 강, partial hills): class stays open (binary projection rule) but defense funnels through the gate sector — sector-level grammar, not region flow. | AGREED 2026-07-07 · TC-④ |
| City (도시) | The coupled spike: one sector where money AND people concentrate — the region's readable target-and-stake point. Separation (초원) is a profile-justified exception. | AGREED 2026-07-07 · TC-⑥ |
| Gate-city (성문 도시) | A city holding the entire border with its rival — every campaign on that front passes through the city sector (하북·한경 vs 중원). | AGREED 2026-07-07 · TC-⑦ |
| Keep city (내륙 심장 도시) | 중원's interior city — invisible from every border; the crown's heart behind a rim of stakes. | AGREED 2026-07-07 · TC-⑦ |
| Port capital (항구 수도) | 동북's capital = its strait sector; the seat's meaning is the island link, and the island holds a knife at it. | AGREED 2026-07-07 · TC-⑦ |
| Battle-summoning placement (전장 소환 배치) | Placement principle: cities are placed where fighting is invited; 서역 is the deliberate opposite pole. | AGREED 2026-07-07 · TC-⑦ |
| Carve principle (조각 원칙) | Post-seal terrain edits are local carves on finished ownership, never candidate/growth re-runs. | AGREED 2026-07-07 · TC-⑫ |
